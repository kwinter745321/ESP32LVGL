# Scope
This is video 95 on a MicroPython LVGL embedded solution. We explain how to build a Smart Touch Pad using a WaveShare ESP32-S3-Touch-LCD-1.69 integrated display.  The display uses a capacitive touch screen; so the display is shiny and the screen is easy to touch.  The whole unit fits into a 3D printed case that can be attached to a laptop.  The menus are dynamic and are editable using a text editor. 

The MCU is an ESP32-S3 N16R8 which means the action is fast and there is plenty of storage.  We provide the latest firmware although the program should operate on any ESP32-S3 firmware with LVGL.  The device communicates across the USB Serial port with a Python program (provided) that interacts with the Smart Touch Pad features. You can fetch the test program from our ESP32LVGL GitHub site. 

In this video, 
 - Look at the WaveShare ST7789 Display and CST816 Touchscreen.
 - Review the display design and data structure of the menu items.
 - Demonstrate the Smart Touch Functions and the Python Desktop program.
 - Review the important parts of the ESP32 and Python code.
