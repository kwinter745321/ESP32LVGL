# Scope
This is video 78 on a SquareLine Studio Tutorial to generate an Arduino LVGL embedded solution. In this video, we walk-through a simple LVGL screen build in SquareLine Studio. We describe our development approach and the hardware.  The Test Rig is a ST7735 and and ESP32 Dev Board. After generating the GUI, we use the Arduino IDE to build and deploy the firmware.  The project uses both LVGL 8.3.11 and Bodmer's TFT_eSPI 2.5.43 libraries that are available at GitHub and through the Arduino Library Manager.

The screens built in SquareLine Studio were modified slightly to make them more appealing.  The newer files are at our GitHub.  The goal was to keep the process simple and describe the overall approach. So we discuss the various steps and then demonstrate the steps in the software.

Note: When switching projects, make sure to set the Preferences (in Arduino) to the SquareLine_Project folder, and ensure you select the current COM port for the deploy.

You can fetch the firmware and programs from our GitHub site, and begin using them immediately.  

In this video, 
 - We describe the Arduino approach.
 - We discuss the Test Rig and wiring.
 - We verify the Test Rig.
 - We create two test GUI screen projects.
 - We build/deploy to Arduino and show the resulting screens.
