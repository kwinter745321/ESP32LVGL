# Scope

This is video 97 on a ESP-IDF embedded solution. We show installing ESP-IDF to build a project (using VS Code) to blink a GPIO pin on an ESP32-S3 board.  We walk you through the subtle steps to configure the environment. We then deploy the firmware onto a typical ESP32-S3 N16R8 board.  You select the target, so one can build Blink for any ESP32 chip.

Actually, we want to use this environment for a future video.  But we thought it best to verify the method first.  This is because some Waveshare WIKI development projects like to use this method (but their instructions are brief.)

You can fetch the firmware files generated during the making of this video from our ESP32LVGL GitHub site. 

In this video, we demonstrate the following steps:
    • We follow the ESP-IDF setup instructions as found on a Waveshare WIKI project.
    • Download of and using the ESP-IDF Installation Manager to build ESP-IDF v5.5.4 environment.
    • Download of and using the Visual Studio Code and setup of the ESP-IDF extension.
    • Selection and configuration of the Blink example.
    • Deploy of the Blink firmware onto an ESP32-S3 target device using esptool.
